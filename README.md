# Typed MongoDB

Type-safe MongoDB operations for Python using Pydantic model introspection.

## Packages

- **typed_mongo**: Runtime package with operators and typed collection wrappers
- **typed_mongo_gen**: Code generator CLI for creating type definitions

## Installation

```bash
pip install typed-mongo           # Runtime only
pip install typed-mongo-gen       # Includes generator CLI
```
